# 更新日志

本项目遵循语义化版本号；`+` 后的数字为构建号。

## [1.1.0] - 2026-07-16

### 新增

- 完整 EPUB 处理链路：结构检查、章节选择、XHTML 分块、批量翻译、质量校验与重新打包。
- 任务历史、断点续译、翻译块缓存、书籍记忆和锁定术语表。
- 翻译前工作量、批次数、耗时和费用预估，以及更清晰的进度、性能与诊断日志。
- Windows 原生文件选择、拖放、输出目录打开与 DPAPI 密钥保护；Android 原生选书、分享与安全存储支持。
- GitHub Actions：在 Windows 上自动执行 `flutter analyze` 与 `flutter test`。

### 改进

- 翻译工作台、设置、任务与预览页面改为统一的桌面优先界面；统一中英文排版、深浅主题与紧凑窗口布局。
- API 提供商预设、连接诊断、翻译参数预设与更完整的错误提示。
- 大型 EPUB 的解包与回写移至 isolate，降低界面阻塞。
- 翻译输出采用同目录临时文件与安全替换；取消或失败时保留已有 EPUB。
- 重试等待与最终打包支持取消；取消后不会提交新的最终输出。
- 日志、任务历史和错误信息会脱敏 API 密钥。

### 测试

- 新增 EPUB 检查/回写、原子输出、取消、重试、缓存、质量校验、原生平台桥接和界面回归测试。
- 提供可选的真实 EPUB 压力测试与真实 API 端到端测试；默认离线测试不消耗 API 额度。

## [1.0.0] - 2026-06-03

- 首个 Flutter 版本发布。
- 支持 Windows 和 Android。
- 提供基础 EPUB 翻译、双语输出、并发翻译和 Material Design 3 界面。
